## Release Notes

### v1.0.0 - Initial Release

**Core Features:**
- AI-powered audiobook generation using Microsoft Edge TTS
- Support for TXT and EPUB formats
- Automatic chapter detection and parsing
- Modern single-page web application
- Real-time progress tracking
- Batch download support

**Technical Stack:**
- Python 3.12+
- FastAPI + Uvicorn
- Vue 3 + TailwindCSS
- Docker + Docker Compose
- Microsoft Edge TTS

**Deployment:**
- Local installation support
- Docker deployment support
- Comprehensive documentation
